# Deployment

Selamat, kamu telah berhasil mempelajari dan mempraktekkan langsung bagaimana membuat aplikasi Python dan machine learning. Tahapan selanjutnya adalah memproduksikan aplikasi yang telah kamu buat. Pada modul ini, kamu akan mempelajari bagaimana cara men-_deploy_ dan me-_release_ aplikasi Python dan machine learning yang telah kamu buat.

Kenapa kamu harus memproduksikan aplikasi Python dan machine learning? Karena dengan memproduksikan aplikasi, kamu bisa membagikan aplikasi kamu kepada pengguna lain. Selain itu, dengan memproduksikan aplikasi, kamu bisa memperlihatkan kemampuan kamu dalam membuat aplikasi Python dan machine learning kepada orang lain. Hal ini tentu akan sangat berguna untuk personal branding dan portfolio kamu.
